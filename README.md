# Кооперативное написание кода

Демо-сайт для совместного написания и выполнения Python кода в реальном времени.

## Возможности

- **Комнаты**: Создавайте или присоединяйтесь к отдельным комнатам для совместной работы
- Совместное редактирование кода с синхронизацией между пользователями в одной комнате
- Подсветка синтаксиса Python
- Нумерация строк с вертикальным разделителем
- Выполнение кода с поддержкой ввода данных
- Консоль для вывода результатов

## Запуск

### Через Docker (рекомендуется)

1. Установите Docker и Docker Compose
2. Клонируйте репозиторий
3. Запустите: `docker-compose up --build`
4. Откройте http://localhost:8000

### Локально

1. Установите Python 3.12+
2. Создайте виртуальное окружение: `python -m venv env`
3. Активируйте: `source env/bin/activate`
4. Установите зависимости: `pip install -r requirements.txt`
5. Запустите: `uvicorn main:app --reload`
6. Откройте http://127.0.0.1:8000

## Использование

- На главной странице введите название комнаты и нажмите "Создать комнату" - система сгенерирует уникальный 4-значный код
- Или введите код существующей комнаты (4 цифры) и нажмите "Присоединиться"
- В комнате отображается название и код
- **Управление скриптами**: В левой панели список скриптов, кликните для переключения
- **Новый скрипт**: Нажмите "+ Новый" для создания нового файла
- Введите Python код в редакторе
- Изменения синхронизируются между всеми подключенными пользователями в этой комнате
- Для выполнения кода нажмите "Запустить код" (запускается текущий скрипт)
- В поле "Входные данные" введите данные для input() (каждое на новой строке)
- Результаты появятся в области "Вывод"
